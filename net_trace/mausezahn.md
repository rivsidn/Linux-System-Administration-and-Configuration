## 简介

报文生成器，用于构造并发送报文。

### 安装

```bash
sudo apt-get install netsniff-ng
```

### 使用

支持两种模式发送报文：

1. 命令行模式
2. 多线程交互模式

命令行模式不支持多线程；

交互模式支持多线程，拥有自己的命令行接口，允许同事创建多个报文，每个报文有不同的参数。



## 命令行模式

### 二进制

二进制模式直接通过二进制固定报文格式。

### 报文类型

通过指定报文类型，激活报文的构造器，在报文类型之后，跟着可选的参数。
