## 基础配置

```bash
# 创建桥
ip link add <bridge-name> type bridge
# 添加桥口
ip link set <eth-name> master bridge-dev
# 查看桥下设备
ip link show master <bridge-name>
# 删除桥口
ip link set <eth-name> nomaster
# 删除桥设备
ip link del <bridge-name> type bridge
```



## vlan配置

设备一共四个口，`eth0` - `eth3` 

```bash
# 桥口的pvid有什么作用？物理口有没有pvid
ip link add bridge-1 type bridge vlan_filtering 1 vlan_protocol 802.1Q vlan_default_pvid 2 vlan_stats_enabled 0
# 将所有设备添加到桥中
ip link set eth0 master bridge-1
ip link set eth1 master bridge-1
ip link set eth2 master bridge-1
ip link set eth3 master bridge-1
```
