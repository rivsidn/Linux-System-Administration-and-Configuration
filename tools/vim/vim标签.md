## 基本操作

| 操作             | 编辑器动作                                               |
| ---------------- | -------------------------------------------------------- |
| m + 字母         | 小写字母为局部标签，文件内有效；<br>大写字母为全局标签。 |
| ` + 字母         | 跳转到指定标签                                           |
| :delmarks + 字母 | 删除标签                                                 |
| :marks           | 查看当前标签                                             |
|                  |                                                          |
