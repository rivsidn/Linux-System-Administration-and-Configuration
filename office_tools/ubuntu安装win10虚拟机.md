## 安装

### 安装GNOME Boxes

```bash
sudo apt install gnome-boxes
```

### 下载Windows 10 ISO

[下载地址](https://www.microsoft.com/en-us/software-download/windows10ISO)

### 安装虚拟机

* 打开`Gnome Boxes`，创建虚拟机，选择下载的`ISO` 文件

* 分配系统资源

* 创建，安装，系统密钥
