### 微信安装

* 添加安装源

  ```bash
  deb http://archive.ubuntukylin.com/ubuntukylin focal-partner main
  ```

* 更新

  ```bash
  apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 56583E647FFA7DE7
  apt update
  ```

* 安装

  ```bash
  apt install weixin
  ```
