`ubuntu` 自带的截图软件虽然方便但是没有编辑功能，写文档过程中会截图不方便标识出具体要关注的位置，所以安装了`flameshot`。



* 软件安装

  ```bash
  sudo apt-get install flameshot
  ```

  

* 快捷键设置

  | Name      | Command                | Shotcut    |
  | --------- | ---------------------- | ---------- |
  | flameshot | /usr/bin/flameshot gui | Ctrl+Alt+l |

  

* 快捷键打开软件

  `Ctrl` + `Alt` + `l`
